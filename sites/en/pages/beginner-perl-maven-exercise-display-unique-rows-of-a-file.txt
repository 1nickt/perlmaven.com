Exercise: display unique rows of a file

=abstract end

Given a the following file:

<code>
A 1
B 2
A 2
C 3
B 2
D 4
C 1 
</code>

Create another file in which the first character is unique:

<code>
A 1
B 2
C 3
D 4
</code>
