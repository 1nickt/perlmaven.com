The <a href="http://www.catalystframework.org/">Catalyst web framework</a>,
is a modern MVC (Model-View-Controller) web framework for Perl. It powers a lot
of commercial sites, a number of <a href="/perl-based-open-source-products">open source products</a>,
and it also powers <a href="http://metacpan.org/">Meta CPAN</a>.
